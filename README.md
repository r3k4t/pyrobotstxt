A simple python program which find out any website robot.txt file.

### Setup[PIP MODULE] ###

+ sudo pip install -r requirements.txt

### Terminal Command ###

+ git clone https://github.com/r3k4t/pyrobotstxt.git
+ cd pyrobotstxt
+ sudo python pyrobotstxt.py
